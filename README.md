# Command-Line Blackjack Game

A classic, interactive Python implementation of Blackjack (21) played directly in the command line. Built using Clean Code principles and Object-Oriented Programming (OOP).

## ✨ Features
- **Object-Oriented Architecture:** Separate classes handling `Card`, `Deck`, `Hand`, and `Chips` logic.
- **Dynamic Ace Scoring:** Program automatically calculates whether an Ace should count as 1 or 11 based on the hand's value.
- **Persistent Betting Bank:** Start with a balance of 100 chips; your chips update dynamically after wins, losses, or pushes.
- **Robust Input Validation:** Prevents crashes from accidental non-integer bets or out-of-bounds plays.
